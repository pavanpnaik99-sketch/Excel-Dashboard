# Excel-Dashboard

This Sales Analysis Dashboard is built in Microsoft Excel to track business performance.  
It provides insights on revenue, orders and customer behavior. The dashboard includes visuals for top products, top cities, monthly revenue trends, category performance and time-based ordering patterns. Slicers allow users to filter by dates and occasions for better decision making.

### Key Insights:
- Total Orders  
- Total Revenue  
- Average Customer Spending  
- Order to Delivery Time  

### Visuals Included:
- Revenue by Occasion  
- Revenue by Category  
- Revenue by Hour  
- Revenue by Month  
- Top 5 Products by Revenue  
- Top 10 Cities by Orders  

### Filters Available:
- Delivery Date  
- Order Date Timeline  
- Occasion Filter  

### Tools Used:
- Excel Pivot Tables  
- Pivot Charts  
- Slicers  
- Basic data cleaning  

### Purpose:
- Identify top selling products  
- Track high performing categories  
- View seasonal and time-based trends  
- Understand customer spending patterns
